# Secure Cloud File Sharing Platform

## Overview

Secure Cloud File Sharing Platform is a full-stack web application that enables users to securely upload, store, manage, and share files using cloud storage. The platform provides authentication, file access control, and secure sharing features.

## Features

- User Registration and Login
- JWT Authentication
- Secure File Upload
- File Download
- File Sharing
- User Dashboard
- Role-Based Access Control
- Cloud Storage Integration
- Activity Logs
